# data-cleaning-in-sql
Practiced data cleaning and preprocessing using SQL in MySQL Workbench


## Overview
Performed data cleaning using SQL in MySQL Workbench by removing duplicate records, handling null values, standardizing inconsistent formatting and incorrect dates, and dropping unnecessary columns. Applied SQL techniques such as DELETE, UPDATE, TRIM, ALTER TABLE, ROW_NUMBER(), joins, and date formatting to improve data consistency and prepare the dataset for further analysis.
